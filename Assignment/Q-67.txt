(67). How can you pick a random item from a range ?

    using of "random.randint()" function i can pick a random item from a range.

    e.g. : 
        l1 = random.randint(1,100)

        print(l1)
---------------------------------------------------------------------------------------

    also i can use of "random.choice()" to  pick a random item from a range.

    e.g. : 
        l1 = range(1.100)
        r1 = random.choice(l1)

        print(r1)
